Features

Home Page

The Home Page contains instructions on how to install and use the application, providing a quick start guide for new users.
  • Logo: Clicking on the logo redirects to the Home page.
  • Hamburger Menu Icon: Clicking on the hamburger menu icon opens a menu containing all available pages.
  • Share Button: Clicking on the share button redirects you to my RobbyTheBot GitHub repository.

Planner Page

The Player Page is used to deploy and monitor missions created on the Planner page.
  • Mission Creation: Users can initiate a mission by entering a name and defining the robot's destination using X and Y coordinates.
  • Mission Table: Created missions are presented in a table format for straightforward viewing and management.

Player Page

The Planner Page allows users to create missions for the robot.
  • Deploy Missions: Users can initiate the created mission.
  • Play Button: Initiates the animation, showing the path taken by the robot.
  • Stop Button: Aborts the mission.
